## Envy, the open OS
Envy is a environment for locked down devices, such as Chromebooks or Windows devices.

## What can Envy do?

  Envy can do multiple things, including:

  * Discord *(via the great people at Discross!)*
  * Windows 2000 *(also via JS/Linux!)*
